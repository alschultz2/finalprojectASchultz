Welcome to Time Tracker

Welcome to my time tracking software. If you have an account already then you can go ahead and log in. If not, go ahead and register for an account! (Your username needs to be at least 5 characters). If you have the manager code you can create a manager account (check readme). With this project, you will be able to log your time spent with a person or doing a task. You can always edit the time later, and the managers can see all users' times.
